# ⚽ Football AI Assistant

## Описание
Чатбот за управление на футболни клубове и играчи с CRUD операции.

## Структура на проекта
- src/ → код (main.py, chatbot.py, clubs_service.py, players_service.py)
- sql/ → schema.sql (структура на базата)
- commands.log → лог на командите
- README.md → този файл

## Стартиране
1. Клонирай репото: `git clone <URL>`
2. Стартирай виртуална среда (по избор)
3. Стартирай чатбота: `python src/main.py`

## Примерни команди
- добави клуб Левски  
- добави играч Меси в Левски позиция FW номер 10  
- покажи играчи на Левски  
- смени номер на Меси на 11  
- изтрий играч Роналдо  
- помощ  
- изход
